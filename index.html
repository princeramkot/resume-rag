🎓 Learn: Core Concepts
📦 CHUNKING

Text is split into overlapping windows (~400 words, 80-word overlap). Overlap ensures context isn't cut at a boundary.

🔢 EMBEDDINGS

Each chunk → 1 536-float vector via text-embedding-3-small. Similar meaning = nearby vectors in space.

🔍 VECTOR SEARCH

Query → embed → cosine similarity vs all chunks. ChromaDB's HNSW index returns top-k results in milliseconds.
